# 第4章：画像処理

## 画像処理タスクの種類

Transformers.jsは様々な画像処理タスクに対応しています：

- **画像分類**: 画像が何であるかを分類
- **物体検出**: 画像内の物体を検出し、位置を特定
- **セグメンテーション**: 画像をピクセル単位で分類
- **画像キャプション**: 画像の内容を文章で説明

## 物体検出（Object Detection）

このチュートリアルでは、物体検出を実装します。物体検出は：
1. 画像内の物体を識別
2. それぞれの物体の位置（バウンディングボックス）を特定
3. 信頼度スコアを提供

## このチュートリアルで学ぶこと

- 画像ファイルのアップロードと処理
- Object Detectionパイプラインの使用
- バウンディングボックスの描画
- 検出結果の視覚化
