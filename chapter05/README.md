# 第5章：音声処理

## 音声処理の種類

Transformers.jsでは様々な音声処理タスクが可能です：

- **音声認識（ASR）**: 音声をテキストに変換
- **音声分類**: 音声の内容や話者を分類
- **音声翻訳**: 音声を別の言語のテキストに変換

## 自動音声認識（ASR）

Automatic Speech Recognition（ASR）は、音声データをテキストに変換する技術です。このチュートリアルでは：
1. マイクからの音声入力
2. 音声ファイルのアップロード
3. Whisperモデルによる高精度な文字起こし

## このチュートリアルで学ぶこと

- 音声ファイルの読み込みと処理
- Automatic Speech Recognitionパイプラインの使用
- マイク入力からのリアルタイム音声認識
- 音声データの前処理
